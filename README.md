The steps for executing is as follows
Download/copy the PowerShell script from https://github.com/nooooooolan/Mathdocs-PDF-Converter
Save it as a Convert-MathcadToPdf.ps1
Move the PowerShell script to the folder with all the Mathcad files you want to convert.
Create a folder in the directory, name it something like "outputs"
Execute the PowerShell script using the following syntax
 https://imgur.com/a/1ftYZEd
It will take like a minute or two, but all your .pdfs should be saved after this!
